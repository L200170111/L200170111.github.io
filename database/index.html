MYSQL

SQL atau juga sering disebut sebagai query merupakan suatu bahasa (language) yang digunakan untuk mengakses database. Secara umum, SQL terdiri dari tiga bahasa, yaitu Data Definition Language(DDL), Data Manipulation Language (DML) dan Data Control Language (DCL).
